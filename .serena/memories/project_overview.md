# 项目概览

## 项目名称
Claude Task Master Visual Interface (claude-task-master-extension)

## 项目目的
为 task-master-ai 项目提供丰富的可视化 VS Code/Cursor 扩展界面。

**核心定位**: 这是对 task-master-ai 的纯粹增强，提供现代化的可视化界面，而不是竞争性替代方案。它需要现有的 task-master-ai 项目，并与 CLI 和 MCP 工具协同工作。

## 主要功能

### 1. 多上下文标签管理 (v1.2.0+)
- 完全支持 Task Master v0.17.0+ 标签格式（兼容至 v0.31.0）
- 标签状态栏：实时当前标签指示器，点击切换功能
- 标签管理命令：直接从 VS Code 命令面板创建、切换、删除和列出标签
- 静默迁移：自动检测和处理旧格式升级，零破坏性变更

### 2. 可视化任务管理
- 层级树视图：以有组织的、可展开的树结构查看所有任务
- 下拉子任务：点击展开/折叠，带有可视化进度指示器
- 智能分组：按状态（待办、进行中、已完成、已阻塞）或优先级查看任务
- 进度概览：实时完成统计和可视化进度条
- 标签感知显示：当前标签上下文和可用标签数量的可视化指示器

### 3. 健壮操作
- CLI 回退系统：MCP 服务器不可用时自动回退到 `task-master-ai` CLI
- 实时更新：任务文件更改时自动刷新
- 跨平台：支持 Windows、macOS 和 Linux
- 错误恢复：优雅处理所有错误场景

## 版本信息
- **当前版本**: 1.3.2
- **发布日期**: 2025-11-01
- **最新特性**: 完整界面中文化

## 依赖要求
1. **Task Master AI**: 需要在项目中设置 task-master-ai
2. **VS Code/Cursor**: VS Code 1.70.0+ 或 Cursor IDE
3. **工作区**: 扩展仅在工作区/文件夹上下文中工作（不支持单个文件）
